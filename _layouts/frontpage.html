Scala in a Nutshell

 click the boxes below to see Scala in action! 

Seamless Java Interop

Scala runs on the JVM, so Java and
Scala stacks can be freely mixed for
totally seamless integration.

Helpful & Expressive Types

Don’t work for the type system.
Let the type system work for you!

Concurrency & Distribution

Run operations on collections in parallel,
use futures for asynchronous programming
or actors for distributed computing.

Traits

Combine the flexibility of Java-style
interfaces with the power of classes.
Think principled multiple-inheritance.

Pattern Matching

A “switch” statement on steroids:
Match against values, objects,
types, collections, and more.

Higher-Order Functions

Functions are first-class objects.
Compose them with guaranteed type safety.
Use them anywhere, pass them to anything.

Extensible

Breathe new life into your existing codebase!
Add new capabilities with context bounds or
extend your types with implicit classes.

Scalable

From small scripts that automate simple tasks
to large mission-critical business applications,
Scala grows with you.

Stable & Mature Ecosystem

Be up and running fast. Get your work done.
Have both first-class Scala libraries as well as
all the platforms' libraries ready at your disposal!

SBT, the Scala Build Tool

would break older clients of your library.
